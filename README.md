# Stack The Odds - My Love2d Game Jam (2026) Submission!

Stack The Odds is a puzzle platformer made for the Love2d Game Jam (2026) where you play as chippy, a special counter with the ability to grow in size by merging with other counters, in this game you'll need to solve puzzles across 3 level with several mechanics built around your ability.
